# platformer-editor

Editor for my online game project, written in TypeScript, HTML and CSS, running in Electron.

### Usage

**npm:**
```
> npm install
> npm run dev
```

**yarn:**
```
> yarn install
> yarn dev
```

### Features

* 2D WebGL Renderer
* UI
* ECS library
* Math library
* 2D collision library
